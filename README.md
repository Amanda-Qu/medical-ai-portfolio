# Medical Imaging AI Portfolio

A dark, image-first portfolio built with Next.js, TypeScript, and Tailwind CSS for medical imaging AI, computer vision, and multimodal AI roles.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deployment

No environment variables are required.

Vercel settings:

- Framework preset: Next.js
- Install command: `pnpm install` or Vercel default
- Build command: `pnpm build` or Vercel default
- Output directory: Next.js default
